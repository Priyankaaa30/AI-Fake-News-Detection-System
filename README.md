# 📰 AI-Powered Fake News Detection System

> Developed by **Priyanka Rajput and team** | Manav Rachna IIRS | Microsoft AI Internship

---

## 🔍 Overview

This project identifies and classifies fake news articles using Machine Learning.
It uses **TF-IDF** for feature extraction and **Logistic Regression** for classification,
wrapped in a fully interactive **Streamlit** web application.

---

## 🚀 Quick Start (Run in 3 steps)

### Step 1 — Install dependencies
```bash
pip install -r requirements.txt
```

### Step 2 — Run the app
```bash
streamlit run app.py
```

### Step 3 — Open in browser
Streamlit will automatically open: **http://localhost:8501**

## 📊 Features

- **Home & Detector** — Paste any news article and get instant Real/Fake prediction

- **About** — Project details, tech stack, references

## 🎯 Learning Objectives

- NLP-based text preprocessing
- Feature extraction using TF-IDF (unigrams + bigrams)
- Supervised learning with Logistic Regression
- Model evaluation: accuracy, confusion matrix, ROC-AUC, F1-score
- Interactive ML web app with Streamlit
